Pilih mode interaksi yang diinginkan melalui "Select Mode"

Draw

Pilih bentuk bidang yang ingin digambar melalui "Select Shape" beserta warna bidang melalui "Select Shape Color"

  • Untuk Line : Klik pada canvas untuk titik pertama, kemudian drag dan klik pada canvas lagi untuk titik kedua
  • Untuk Square : Klik pada canvas untuk titik pertama, kemudian drag dan klik pada canvas untuk menggambar persegi
  • Untuk Rectangle : Klik pada canvas untuk titik pertama, kemudian drag dan klik pada canvas untuk menggambar persegi panjang
  • Untuk Polygon : Pilih jumlah sudut dan jenis mode penggambaran (parametrik atau per garis), kemudian gambar melalui canvas

Change Color

Pilih warna baru melalui "Select Shape Color", lalu klik pada bidang yang ingin diubah warnanya

Move Point

Klik pada titik sudut yang ingin digeser, kemudian drag dan klik pada titik baru

Move Shape

Klik pada bidang yang ingin digeser, kemudian drag dan klik pada tempat baru

Save and Load

Untuk menyimpan hasil gambar, klik tombol "Save" dan file data gambar saat ini akan diunduh secara otomatis

Untuk me-load hasil gambar yang sudah disimpan sebelumnya, klik tombol "Choose File" dan pilih file yang telah diunduh sebelumnya

Clear Canvas

Untuk membersihkan canvas, klik tombol "Clear Canvas"
